Principios del sistema
🎯

Un solo tag de posición activo

En todo momento el contacto tiene exactamente UN tag de posición activo. Al avanzar de paso, se borra el anterior y se añade el nuevo.

📚

Historial permanente

Los tags de historial y de tier nunca se borran. Se acumulan para registrar qué ocurrió con el lead.

⚙️

Orden universal en cada step

1. Stop Bot (si aplica) → 2. Remove Tag anterior → 3. Add Tag nuevo → 4. Acción del negocio. Siempre en este orden.

🚧
